# Global Internet Outage Analyzer — Spark ML

A Big Data Analytics project that collects global internet outage data, 
processes it with Apache Spark, predicts outage risk per country using 
Random Forest, and visualizes results on an interactive world dashboard.

## Tech Stack
- Apache PySpark
- scikit-learn (Random Forest)
- Plotly Dash
- IODA API
- Open-Meteo API
